# Setup Scripts

These scripts create a local `.venv` on each machine and install `requirements.txt`.

Mac mini:

```bash
bash scripts/setup_mac.sh
```

Lab PC Windows PowerShell:

```powershell
.\scripts\setup_lab_pc_windows.ps1
```

Lab PC Linux:

```bash
bash scripts/setup_lab_pc_linux.sh
```

The scripts require Python 3.10 to already be installed.
